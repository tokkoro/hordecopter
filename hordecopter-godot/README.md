# Hordecopter (Godot)

Hordecopter is a small Godot project. This repository contains the game scene files, scripts, and assets needed to run the project in the Godot editor.

## Requirements

- [Godot Engine](https://godotengine.org/) (project is set up for Godot 4.x)

## Getting Started

1. Open the Godot project manager.
2. Click **Import** and select this folder.
3. Open `theworld.tscn` (or `hordecopter.tscn`) and press **Play**.

## Project Layout

- `project.godot` - Godot project configuration.
- `*.tscn` - Scenes.
- `*.gd` - GDScript files.
- `*.glb`, `*.png`, `*.obj` - 3D models and textures.
